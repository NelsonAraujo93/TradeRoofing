# 📖 Trade Roofing <a name="trade-roofing"></a>

**Trade Roofing** This is a MERN project that allows to kind of users, Manager and Costumer. The managers have a personal dash board to analyse and perform authorized functionalities, such as update, delete, create.
In the other hand the user can manage only its properties and his services requests.
